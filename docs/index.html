XMP ↔ NP3 변환기

Adobe Lightroom / Camera Raw 프리셋(.xmp)과 니콘 카메라용 Flexible Picture Control(.NP3)을 양방향 변환합니다. 확장자를 보고 방향을 자동으로 정하며(.xmp → .NP3, .NP3 → .xmp), 여러 파일을 한 번에 올릴 수 있고, 모든 처리는 브라우저 안에서만 이루어집니다 — 파일이 서버로 전송되지 않습니다.

⤓

XMP 또는 NP3 파일을 여기에 드래그하거나 클릭해서 선택

여러 개 동시 변환 가능 · .xmp → .NP3 · .np3/.ncp → .xmp · JPG/PNG를 올리면 미리보기 샘플 사진으로 사용
XMP → NP3 — 톤(대비·하이라이트·섀도·화이트·블랙), 채도(+바이브런스 절반 반영), HSL 8밴드 → 컬러 블렌더, 스플릿 토닝/컬러 그레이딩, 샤프닝·명료도·텍스처(→미드레인지), 노출·파라메트릭·마스터 톤커브(→ 257포인트 휘도 커브로 베이크)가 변환됩니다. 화이트밸런스·R/G/B 채널별 커브·디헤이즈·그레인은 NP3 포맷에 슬롯이 없어 제외됩니다.
NP3 → XMP — NP3가 담고 있는 모든 항목(톤 슬라이더, 컬러 블렌더 → HSL, 컬러 그레이딩 → 스플릿 토닝, 톤커브 앵커, 샤프닝·명료도·미드레인지)이 Lightroom 프리셋으로 1:1 복원됩니다. 구형 51바이트 NCP는 기본 슬라이더만 옮겨집니다.
미리보기 — 변환된 행에 마우스를 올리면 샘플 사진에 프리셋을 실시간 적용해 보여줍니다(왼쪽: 변환 전 렌더/원본, 오른쪽: NP3 결과 — 경계선에서 변환 손실을 비교). JPG/PNG 파일을 드롭하면 내 사진으로 미리볼 수 있고, 사진도 서버로 전송되지 않습니다.
사용법 — 변환된 .NP3는 SD카드의 NIKON/CUSTOMPC 폴더에 넣고 카메라 메뉴 → Picture Control 관리 → 메모리카드에서 복사 (Z6III · Z5II · Z50II · Z8 · Z9 · Zf 등 Flexible Picture Control 지원 기종과 NX Studio). 변환된 .xmp는 Lightroom의 프리셋 가져오기로 불러옵니다.
행에 마우스를 올리면 적용 결과를 미리봅니다 · JPG/PNG 드롭 = 샘플 사진 교체
NPC Simulator 프로젝트의 변환 엔진을 그대로 사용합니다.
